This is a web app designed especially for use of farmers all around the world. It provides the user with real time weather data for their reapective location.
The webapp basically asks the farmer to enter "Crop Name" where he/she needs to enter the name of crop they want to check about, "Type of Crop" i.e. Rabi, Kharif or Zaid, and the "City". Then the app fetches the weather data using OPEN-WEATHER API and compares it with the previously saved data about crops and gives the result. Result contains solution to the problem if any found related to humidity, temperature etc w.r.t. to crops else it returns NO ISSUES FOUND prompt.
The webapp is designed to be minimalistic and simple so that anyone can access it from anywhere.
